How to Use the App

The user interacts with the app by adding locations where gravel liquefaction has occured. The data added by the user is added to a table and the map shown on the home page.

Adding Data

To add data to the table and the map the user must navigate to the "Add Data" page. Here the user will have the option to add the data. Once the data is added the table and the map will update to reflect the changes.
